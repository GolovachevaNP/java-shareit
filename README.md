# java-shareit

Template repository for ShareIt project.

![ER-диаграмма ShareIt](ER-diagram%20ShareIt.jpg)

## Структура базы данных

### Основные сущности

- `users` - пользователи
- `items` - вещи, которые пользователи могут сдавать в аренду
- `bookings` - бронирования вещей
- `comments` - отзывы пользователей о вещах
- `requests` - запросы пользователей на добавление нужных вещей

### Связи между таблицами

- `items.owner_id` - владелец вещи, связь с таблицей `users`
- `items.request_id` - запрос, по которому добавлена вещь, связь с таблицей `requests`
- `bookings.item_id` - бронируемая вещь, связь с таблицей `items`
- `bookings.booker_id` - пользователь, который бронирует вещь, связь с таблицей `users`
- `comments.item_id` - вещь, к которой оставлен отзыв, связь с таблицей `items`
- `comments.author_id` - автор отзыва, связь с таблицей `users`
- `requests.requestor_id` - автор запроса, связь с таблицей `users`

## Таблицы

### `users`

- `id` - уникальный идентификатор пользователя
- `name` - имя пользователя
- `email` - электронная почта пользователя

### `items`

- `id` - уникальный идентификатор вещи
- `name` - название вещи
- `description` - описание вещи
- `is_available` - доступность вещи для аренды
- `owner_id` - идентификатор владельца вещи
- `request_id` - идентификатор запроса, по которому добавлена вещь

### `bookings`

- `id` - уникальный идентификатор бронирования
- `start_date` - дата и время начала бронирования
- `end_date` - дата и время окончания бронирования
- `item_id` - идентификатор бронируемой вещи
- `booker_id` - идентификатор пользователя, который бронирует вещь
- `status` - статус бронирования

### `comments`

- `id` - уникальный идентификатор отзыва
- `text` - текст отзыва
- `item_id` - идентификатор вещи
- `author_id` - идентификатор автора отзыва
- `created` - дата и время создания отзыва

### `requests`

- `id` - уникальный идентификатор запроса
- `description` - описание нужной вещи
- `requestor_id` - идентификатор автора запроса
- `created` - дата и время создания запроса
